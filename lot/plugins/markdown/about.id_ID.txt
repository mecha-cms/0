> Konversi pola Markdown menjadi HTML.

**Markdown** adalah dua hal: sintaks teks markup polos, dan perangkat lunak yang mengubah markup teks biasa ke HTML untuk penerbitan di web.

Sintaks Markdown memungkinkan Anda untuk menulis teks secara alami dan memformatnya tanpa menggunakan tag HTML. Lebih penting lagi: dalam format Markdown, teks Anda tetap menyenangkan untuk dibaca bagi manusia. Jika Anda menggunakan email berformat teks, Anda sudah tahu beberapa bagian dari sintaks yang ada.

*[HTML]: Hyper Text Markup Language
