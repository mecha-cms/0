<p>﻿Pastikan berkas plugin yang akan Anda unggah memiliki struktur konten seperti ini:</p>
<pre><code class="no-highlight">plugin.zip
└── plugin/
    ├── file-1.txt
    ├── file-2.txt
    ├── file-3.txt
    ├── file-4.txt
    └── …</code></pre>
<p>Anda perlu menambahkan sebuah folder akar untuk membungkus berkas-berkas plugin Anda, dan kemudian Anda perlu menamai berkas <code>ZIP</code> tersebut dengan nama yang sama seperti nama folder tersebut.</p>
