<p>﻿Pastikan berkas tameng yang akan Anda unggah memiliki struktur konten seperti ini:</p>
<pre><code class="no-highlight">shield.zip
└── shield/
    ├── file-1.txt
    ├── file-2.txt
    ├── file-3.txt
    ├── file-4.txt
    └── …</code></pre>
<p>Anda perlu menambahkan sebuah folder akar untuk membungkus berkas-berkas tameng Anda, dan kemudian Anda perlu menamai berkas <code>ZIP</code> tersebut dengan nama yang sama seperti nama folder tersebut.</p>
<p>Setelah itu, Anda dapat mengaktifkan tameng melalui <a href="{{url.manager:config}}" title="Konfigurasi">manajer konfigurasi</a> atau dengan menekan tombol <strong>Pasang</strong>.</p>
