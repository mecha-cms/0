> Secara otomatis menyisipkan tag meta [Facebook Open Graph](https://developers.facebook.com/docs/sharing/opengraph "Open Graph Stories") ke dalam kepala **Mecha**.

Menambahkan tag meta **Open Graph** ke dalam `<head>` pada halaman web HTML Anda akan memberikan _crawler_ info yang terstruktur seperti judul, deskripsi dan gambar thumbnail untuk konten.

*[HTML]: Hyper Text Markup Language
