> Reduce page size by detracting the HTML output.

This plugin helps you to dispose some parts of the HTML markup in the page, which is actually not going to affect anything if you omit them, such as whitespaces and root domain in the URL.

*[HTML]: Hyper Text Markup Language
*[URL]: Uniform Resource Locator
